Follow these 5 steps to wire up Gmail. Takes about 5 minutes. Once done, every application is emailed directly to benbaxventures@gmail.com.

1
Create a free EmailJS account

Go to emailjs.com → click Sign Up Free → register with your Gmail.

2
Connect your Gmail as an Email Service

Email Services tab → Add New Service → choose Gmail → authorise benbaxventures@gmail.com → Create Service. Copy the Service ID (e.g. service_abc1234).

3
Create an Email Template

Email Templates tab → Create New Template → set Subject to New Application – {{applicant_name}} → switch Content to HTML editor, delete everything, paste the template below → Save. Copy the Template ID (e.g. template_xyz5678).


          

        

      


      

        4
        

          Get your Public Key
          
Click your Account avatar (top-right) → General tab → copy your Public Key (e.g. AbCdEfGhIjK123).

        

      


      

        5
        

          Paste the 3 values into this file & host it
          
Open benbax_internship_application.html in VS Code → scroll to the bottom → find the EMAILJS CONFIGURATION block → replace YOUR_PUBLIC_KEY, YOUR_SERVICE_ID, and YOUR_TEMPLATE_ID with your real values → save → this panel disappears and the form is live.
